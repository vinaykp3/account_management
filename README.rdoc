####redis setup instruction ######

1. wget http://download.redis.io/releases/redis-2.8.17.tar.gz
2. tar xzf redis-2.8.17.tar.gz
3. cd redis-2.8.17
4. sudo make install
5. cd utils
6. sudo ./install_server.sh

#### redis server will be running  #####
